Fórmulas resumidas

Cobertura

Cobertura (%) = mínimo entre 100 e (metros de segurança / comprimento da correia) × 100.

Conversão dos segmentos

Exemplos: “4 x 500” = 2000 m; “1 x 482 + 1 x 454 + 1 x 402” = 1338 m.

Score de risco

A = 35 pontos, B = 22, C = 10, N/A = 15. Soma-se penalização por baixa cobertura, quantidade de ordens SAP, status e faixa de valor.

Período analisado

O período SAP é lido das datas presentes na importação atual. Esse período aparece no topo e acompanha a base selecionada.

Como explicar ao gerente

Este painel foi desenhado para leitura rápida. Ele consolida duas planilhas padrão, recalcula automaticamente a base e mostra o mesmo método em todos os gráficos, tabelas e mapas. Quando uma nova planilha é importada, o sistema registra a data da carga, recalcula o período SAP e passa a exibir os resultados daquela base selecionada.

Como usar

  1. Importe INFO TEC CORREIAS - SAMARCO.xlsx.
  2. Importe SAP CORREIA.xlsx.
  3. Quando as duas planilhas forem escolhidas na sessão, o sistema consolida a carga pareada e atualiza filtros, KPIs, gráficos, mapa, tabelas e explorador de abas.
  4. Novas importações funcionarão bem mantendo os mesmos nomes de abas e a mesma estrutura-base das planilhas atuais.

A base já abre carregada com os dados atuais para facilitar demonstração imediata.
